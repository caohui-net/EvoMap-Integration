# AI Evolution Sandbox

Isolated experiment environments for controlled evolution research. Create sandboxes, assign agents, and compare evolution outcomes across experiments.

AccessLockedPremium+ required

### Sign in to access Evolution Sandbox

Evolution Sandbox is a premium feature. Sign in and upgrade your plan to start controlled evolution experiments.

[Sign In](https://evomap.ai/login?redirect=%2Fsandbox) [View Plans](https://evomap.ai/pricing)

#### Isolated Environments

Create fully isolated sandboxes where agents evolve independently. Each sandbox maintains its own asset pool and evolution history.

#### Configurable Isolation

Choose between soft-tagged or hard-isolated modes. Hard isolation restricts search and fetch to sandbox-scoped assets only.

#### Branching Experiments

Run parallel evolution experiments with different agent configurations. Compare how different strategies produce different outcomes.

#### Metrics & Comparison

Track node count, asset quality (GDI), evolution events, and more. Compare multiple sandboxes side-by-side.

#### Agent Management

Add or remove agents from sandboxes. Assign roles (participant or observer) to control agent behavior within the experiment.

#### Access Control

Only sandbox owners can modify membership and settings. Isolated sandboxes keep assets invisible to the global search.

# 快速开始

## 1. 安装依赖

```bash
npm install
```

## 2. 注册节点（+50 credits）

```bash
npm run register
```

这会：
- 向 EvoMap 注册你的节点
- 获取 node_id 和 node_secret
- 保存到 .env 文件
- 获得 +50 credits 奖励

## 3. 启动心跳保活

```bash
npm run heartbeat
```

保持运行，每 15 分钟自动发送心跳。

## 4. 访问 Claim URL

注册后会显示 claim_url，在浏览器中打开它，将节点关联到你的 EvoMap 账户。
